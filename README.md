# BANILACO SQUAD

> **습관이 성과가 되고, 성과가 명예가 되는 크리에이터 주도 성장 엔진**
>
> TikTok Shop 어필리에이트 크리에이터를 위한 K-Beauty 브랜드 성장 플랫폼.
> 단순 커미션 지급 구조가 아닌, 크리에이터의 자발적 참여를 이끄는 **심리학 기반 후킹 시스템**.

---

## Part 1 — 왜 이 시스템을 만들었는가?

### 크리에이터 프로그램의 고질적 문제 3가지

| 문제 | 증상 | 우리 해법 |
|------|------|-----------|
| **① 초기 이탈** | 가입해도 첫 주에 70% 이탈 | Daily Mission + Onboarding Checklist로 첫 3일 동안 고정 루틴 제공 |
| **② 중기 정체** | Rose 티어 찍으면 성장 둔화 | Public Countdown + Mystery Mission으로 "다음 목표"를 항상 보이게 |
| **③ 엔드게임 공허** | 최상위 크리에이터 동기 상실 | PINK LEAGUE 시즌제 + 구간별 carry-over 보상 |

### 기술이 아닌 **행동 경제학**으로 푸는 문제

| 심리학 원리 | 적용 기능 |
|------------|----------|
| **Habit Loop** (습관 고리) | Daily Mission + Streak |
| **Goal Gradient Effect** (목표 근접 가속) | Public Countdown ("Rose까지 7 missions 남음") |
| **Variable Ratio Reinforcement** (슬롯머신 원리) | Mystery Mission (1x~5x 랜덤) |
| **Loss Aversion** (손실 회피) | Streak Soft Reset (완전 리셋 대신 한 단계 하강) |
| **Social Proof** (동조) | 공개 티어 진행률, Discord 랭킹 |
| **Sunk Cost + Carry-over** (누적 자산) | 시즌 구간별 다음 시즌 멀티플라이어 |

## Part 2 — 크리에이터 여정 (Pink Squad 2.0)

```
🌸 Pink Petal  →  🌹 Pink Rose  →  💎 Pink Diamond  →  👑 Pink Crown
   (입문)           (성장)            (전문)              (엘리트)
```

| 티어 | 조건 | 커미션 | Squad 보너스 | 특징 |
|------|------|--------|-------------|------|
| 🌸 **Pink Petal** | 가입 즉시 | 10% | 0% | 학습 + 첫 콘텐츠 |
| 🌹 **Pink Rose** | 미션 50개 OR 월 GMV $500 | 12% | 2% | 창작 챌린지 |
| 💎 **Pink Diamond** | 미션 200개 OR 월 GMV $2,500 | 15% | 3% | PINK LEAGUE 참가 가능 |
| 👑 **Pink Crown** | 월 GMV $10,000 OR 리그 우승 | 18% | 5% | 브랜드 앰버서더급 |

**핵심 설계 의도**: 티어는 "현재 실력"이 아니라 "지금까지의 투자"를 보여준다. 한번 올라간 티어는 자동 하락하지 않음 → 손실 회피 심리 활용.

## Part 3 — 6가지 후킹 메커니즘

### 🎯 Hook #1: Daily Mission + Streak

> **한 줄 설명**: 매일 1개 이상 미션 완료하면 보상이 점점 커진다.

**작동 방식:**
- 3가지 미션 타입: Learning(📚), Creation(🎬), Viral(🚀)
- 연속 완료 일수에 따라 리워드 멀티플라이어 증가

| 스트릭 | 멀티플라이어 | 심리 효과 |
|--------|------------|----------|
| 3일 | 1.2x | 🔥 On Fire |
| 7일 | 1.5x | ⚡ Week Warrior |
| 14일 | 1.8x | 💪 Unstoppable |
| 30일 | 2.0x | 👑 Legend |

**⭐ 소프트 리셋** (Duolingo Streak Freeze 원리):
- ❌ 기존: 하루 빠지면 30일 → 1일로 리셋 → "그냥 포기"
- ✅ 우리: 하루 빠지면 30일 → 14일로 한 단계만 하강
- 2일 빠지면 두 단계 하강, 3일+ 빠지면 완전 리셋

**왜 중요한가**: 스트릭이 깨져도 "14일부터 다시"가 "1일부터 다시"보다 복귀율이 훨씬 높음.

---

### 📊 Hook #2: Public Countdown

> **한 줄 설명**: "Rose까지 7 missions 남음"을 항상 보이게 해서 포기할 수 없게 만든다.

**작동 방식:**
- 대시보드 상단에 상시 표시
- Discord 프로필, 커뮤니티 랭킹에도 노출 예정
- 원형 진행률 위젯 + "거의 다 왔어요!" 카피

**두 가지 심리 효과 동시 발생:**
1. **본인**: Goal Gradient Effect — 목표에 가까울수록 가속 (스타벅스 앱 "별 3개만 더 모으면 무료 음료" 원리)
2. **주변**: Social Proof — "쟤 곧 승급이네, 나도 해야지"

---

### 🎰 Hook #3: Mystery Mission (Variable Reward)

> **한 줄 설명**: 하루 1개 "?" 미션. 열어봐야 보상이 1x~5x 중 뭘지 안다.

**확률 분포:**

| 멀티플라이어 | 확률 | 라벨 |
|------------|------|------|
| 1x | 50% | 🎯 Normal |
| 2x | 25% | ✨ Double |
| 3x | 15% | 🔥 Triple |
| 4x | 8% | 💎 Mega |
| 5x | 2% | 🎰 **JACKPOT** |

**기대값: 약 1.87x** (평균 1배 미션 대비 비용 증가 87%)

**왜 이 투자가 수지 맞는가:**
- 매일 앱 여는 이유 만듦 ("오늘 뭐 나올지 궁금해")
- 슬롯머신의 중독성 원리를 윤리적으로 활용
- Jackpot(4x+) 터지면 크리에이터가 자발적으로 SNS에 자랑 → 바이럴

---

### 🏆 Hook #4: PINK LEAGUE (4주 시즌제)

> **한 줄 설명**: Diamond 이상만 참가, 실시간 랭킹, 시즌 종료 시 전원 보상.

**Pink Score 계산:**
```
Pink Score = (월 GMV × 1.0) + (바이럴 인덱스 × 0.5)
바이럴 인덱스 = (조회 0.3 + 공유 0.4 + 좋아요 0.2 + 댓글 0.1) / 팔로워 × 1000
```

**⭐ 시즌 종료 시 구간별 carry-over 보상** (TOP 10만 보상받는 구조의 함정 해결):

| 최종 순위 | 다음 시즌 시작 멀티플라이어 | 의미 |
|----------|-------------------------|------|
| TOP 10% | **1.5x** | Elite |
| TOP 30% | **1.3x** | Top Performer |
| TOP 50% | **1.2x** | Rising Star |
| 전원 (참가자) | **1.1x** | Participation |

**왜 중요한가**: "졌지만 잃지 않은" 경험을 만들어야 다음 시즌도 참가함. 시즌 간 공백도 자동 해결.

---

### 🤝 Hook #5: Collab Duo (수평적 협력)

> **한 줄 설명**: 두 크리에이터가 같은 제품으로 콜라보하면 Pink Score에 부스트.

**보상 구조:**

| 조건 | 부스트 | 비고 |
|------|--------|------|
| 기본 콜라보 | +15% | verified 상태 |
| 리그 시즌 중 | +20% | 더 강한 유인 |
| Dynamic Duo (3회 연속) | +5% 추가 | 팀워크 보너스 |

**제한 (exploit 방지):**
- 주당 최대 3개 콜라보
- 시즌당 같은 파트너 최대 5회

**왜 이게 차별점인가**: 기존 크리에이터 프로그램은 전부 "개인 플레이". 서로 파이 뺏는 구조가 아니라 **같이 키우는** 구조로 재설계.

---

### 👥 Hook #6: Squad 수익공유

> **한 줄 설명**: 내가 초대한 크리에이터의 GMV 일부가 나한테 패시브로 들어온다.

**수익 공유율:**

| 나의 티어 | Squad 보너스 |
|----------|-------------|
| 🌸 Petal | 0% |
| 🌹 Rose | 2% |
| 💎 Diamond | 3% |
| 👑 Crown | 5% |

**핵심 설계**: 다단계 구조가 아닌 1단계만 공유 (pyramid scheme 아님). 내가 데려온 직계 멤버만.

## Part 4 — 안전장치 (돈 새지 않게)

### 🛡️ Zombie Filter (CHS = Creator Health Score)

> **가장 중요한 안전장치**. 봇팜 / 좀비 계정이 미션만 돌려서 돈을 뽑아가는 걸 구조적으로 차단.

**CHS 계산 공식:**
```
CHS = (Sales 40%) + (Content 30%) + (Engagement 20%) + (Mission Diversity 10%)
```

**4개 존 자동 분류:**

| Zone | CHS | 현금 리워드 | 스트릭 캡 | 일일 미션 제한 |
|------|-----|-----------|----------|--------------|
| 🟢 GREEN | 70+ | 100% | 무제한 | 무제한 |
| 🟡 YELLOW | 40-69 | 100% | 1.2x 캡 | 무제한 |
| 🟠 ORANGE | 20-39 | **50%** | 1.2x 캡 | **2개/일** |
| 🔴 RED | 0-19 | **0% (스코어만)** | 1.0x | — |

**신규 크리에이터 보호**: 가입 14일 이내는 CHS 최소 40 (YELLOW) 보장 → 정상 신규 가입자가 RED에 걸리지 않음.

**봇팜 시나리오 대응:**
```
봇 계정: GMV=0, 콘텐츠=0, 참여율=0
→ CHS = 0 + 0 + 0 + 3 = 약 3점
→ 자동 RED → 현금 지출 0
→ 스코어만 쌓이고 현금은 안 나감
```

### 🛡️ Anti-Exploit Rules

미션 완료 시 병렬 실행되는 4가지 필터:

| 룰 | 기준 | 차단 조치 |
|----|------|----------|
| Velocity | 60분 내 5개 초과 완료 | 429 Too Many Requests |
| Proof URL 재사용 | 같은 URL 다른 미션에 사용 | 거부 |
| Account Age Burst | 3일 이내 + 10개 초과 | 차단 |
| 당일 중복 Proof | 같은 URL 같은 날 | 거부 |

## Part 8 — 기술 정보 (개발자용)

### Stack

| 영역 | 기술 |
|------|------|
| Frontend | Next.js 16.2 (Turbopack) + TypeScript 5.9 + Tailwind CSS 4 |

```
src/
├── lib/
│   ├── streak/           # 스트릭 엔진 (소프트 리셋)

│   ├── health/           # CHS + Anti-Exploit

│   ├── squad/            # Squad 수익공유
│   └── discord/          # Discord 봇 알림

│   ├── creator/          # 크리에이터 포털
```

```bash
# 1. Install

# 2. Env
cp .env.example .env.local

bun run dev
```

## 요약: 이 시스템의 핵심 사상

> **"합리적 참여"가 아닌 "빠져드는 참여"를 설계했다.**

- 예측 가능한 보상만 있는 프로그램 → 합리적이지만 중독은 없음
- 우리가 추가한 것: **불확실성(Mystery) + 공개 진행률(Countdown) + 손실 완화(Soft Reset + Carry-over)**
